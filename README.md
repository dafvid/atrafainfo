# atrafaweb

## Setup
1. Install Homebrew
2. `brew install python`
3. `git clone git@github.com:dafvid/atrafaweb.git`
4. `cd atrafaweb`
5. `python3 -m venv venv`
6. `venv/bin/pip install -U pip`
7. `venv/bin/pip install -U wheel`
8. `venv/bin/pip install -r requirements.txt`

## Create test data
1. `python3 test.py`
